# Arkanoid
> Tested on Linux

## Author
Patryk Ilkiw

## Description

This is a simple Arkanoid game made in Python in terminal.

## How to play

1. Run `python3 arkanoid.py`
2. Use the left and right arrow keys to move the paddle.
3. `<Ctrl>+C`x2 to close or in endgame screen press any key
4. To restart `<Ctrl>+C`
